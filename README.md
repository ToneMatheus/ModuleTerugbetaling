# Technisch - module Voor Terugbetaling

Binnen Ventigrate worden er wel veel zaken voorgeschoten; Quick, pizza hut, broodjes, drankautomaat, enz. Rekeningen worden betaald door 1 persoon die dan bij iedereen moet langs gaan om af te rekenen.

Handig zou zijn mocht er een applicatie worden gemaakt waar deze kosten kunnen bijgehouden worden. Wekelijks wordt er dan een melding gemaakt van de totaal openstaande kosten (en bij wie) en worden de nodige herinneringen uitgestuurd. Deze applicatie zou na de stage daadwerkelijk worden gebruikt.

Extra mogelijke features zijn:

* Linken met een bruikbare betalingslink (eventueel via SMS) of betalingssysteem (Payconiq)
* Bestellingsmodule voor broodjes
* Integratie in MS Teams
